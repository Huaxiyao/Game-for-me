# 🎮 Game-for-me 小游戏合集

一个可直接在浏览器中运行的小游戏合集，无需安装任何依赖。打开即玩。

▶️ **在线游玩：https://huaxiyao.github.io/Game-for-me/**

## 游戏列表

| # | 游戏 | 文件 | 玩法 |
|---|------|------|------|
| 1 | 🧠 舒尔特方格 | `舒尔特方格.html` | 按顺序点击 1→N，训练注意力 |
| 2 | 🧠 舒尔特方格 Pro | `舒尔特方格Pro.html` | 多尺寸、Chart.js 数据图表、主题切换 |
| 3 | ⚡ 反应速度测试 | `反应速度测试.html` | 红色变绿色的瞬间点击，测反应时间 |
| 4 | ⚫ 五子棋 | `五子棋.html` | 双人轮流落子，五子连珠获胜 |
| 5 | 🖱️ 手速测试 (CPS) | `手速测试.html` | 10秒/30秒/60秒内疯狂点击，测 CPS |
| 6 | 🃏 记忆翻牌 | `记忆翻牌.html` | 翻牌匹配，考验短期记忆力 |
| 7 | ⌨️ 打字速度测试 | `打字速度测试.html` | 输入给定文本，测 WPM 和准确率 |
| 8 | 💣 扫雷 | `扫雷.html` | 经典扫雷，左键翻开右键标旗 |
| 9 | 🔢 数字记忆 | `数字记忆.html` | 记忆数字序列，逐级递增难度 |
| 10 | 🧩 数独 | `数独.html` | 自动生成谜题，简单/中等/困难 |


## 特色

- **无依赖** — 所有游戏均为单个 HTML 文件，打开即玩
- **双模式** — 支持鼠标/触屏和键盘（空格键）操作
- **响应式** — 适配手机、平板、桌面端
- **本地记录** — 成绩保存在浏览器 localStorage 中
- **GitHub Pages** — 直接在线访问，无需下载

## 使用方式

**在线访问**：https://huaxiyao.github.io/Game-for-me/

**本地运行**（克隆仓库）：
```
git clone https://github.com/Huaxiyao/Game-for-me.git
```
然后用浏览器打开任意 `.html` 文件。
