# 模式切换功能测试指南

## 测试环境
在浏览器中打开 `typing-test.html` 文件

## 测试步骤

### 1. 按文本打字模式测试
1. 确认当前模式为"按文本打字"（按钮高亮）
2. 点击"开始测试"按钮或按 Enter 键
3. 输入显示的文本
4. **验证点**：
   - 输入完文本后应自动结束
   - 显示已用时间（从 0s 递增）
   - 统计信息正确显示

### 2. 60秒计时模式测试
1. 点击"60秒计时"按钮切换模式
2. 确认模式切换成功（按钮高亮）
3. 点击"开始测试"按钮或按 Enter 键
4. **验证点**：
   - 初始显示 60s
   - 倒计时从 60s 递减到 0s
   - 60秒后自动结束
   - 统计信息正确显示

### 3. 随机单词模式测试
1. 点击"随机单词"按钮切换模式
2. 确认模式切换成功（按钮高亮）
3. 点击"开始测试"按钮或按 Enter 键
4. 输入显示的随机单词
5. **验证点**：
   - 输入到文本末尾时应自动扩展更多单词
   - 显示已用时间（从 0s 递增）
   - 按 Enter 键可重新开始测试
   - 统计信息正确显示

### 4. 模式切换测试
1. 在任意模式下开始测试
2. 点击其他模式按钮切换模式
3. **验证点**：
   - 切换模式后应重置状态
   - 输入框清空
   - 统计信息重置
   - 可以正常开始新测试

## 常见问题排查

### 问题1：倒计时模式不显示剩余时间
- 检查是否正确切换到"60秒计时"模式
- 确认点击了"开始测试"按钮

### 问题2：随机单词模式不自动扩展文本
- 确认输入到文本末尾（还剩约5个字符时应自动扩展）
- 检查浏览器控制台是否有错误信息

### 问题3：Enter 键不能重新开始
- 确认当前是"随机单词"模式
- 确认测试已经开始（输入框可输入）

### 问题4：模式切换后状态未重置
- 检查是否点击了模式按钮（而非其他区域）
- 确认按钮高亮状态已改变

## 技术说明

### 修复内容
1. **countdown 模式显示剩余时间**：倒计时从 60s 递减到 0s
2. **random 模式自动扩展文本**：输入到还剩5个字符时自动添加更多单词
3. **random 模式 Enter 键重新开始**：打字过程中按 Enter 可重新开始
4. **countdown 模式初始显示**：开始时显示 60s

### 代码修改位置
- `updateStats()` 函数：countdown 模式显示剩余时间
- `checkComplete()` 函数：random 模式自动扩展文本
- `start()` 和 `resetState()` 函数：countdown 模式初始显示 60s
- Enter 键事件处理：random 模式下重新开始
