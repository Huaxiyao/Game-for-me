# 打字速度测试 · 重写设计文档

## 概述

重写 `typing-test.html`，保留核心打字概念，重新设计整个体验。采用极简专注风格（类似 Monkeytype），单文件架构，核心功能优先，高级功能渐进展示。

## 架构方案

**方案 A：极简单文件 · 核心优先**

- 单个 HTML 文件，符合项目惯例
- 核心打字体验优先实现
- 高级功能（历史、排行榜、自定义文本）作为可折叠面板
- 预估代码量：400-500 行

## 功能清单

### 核心功能

1. **中英文切换**：顶部按钮切换，切换后重置当前状态
2. **WPM + 准确率**：实时计算并显示
   - 中文：字/分钟（正确字符数 / 时间）
   - 英文：WPM（正确字符数 / 5 / 时间）
3. **实时反馈**：
   - 正确字符：绿色 `#10b981`
   - 错误字符：红色 `#ef4444` + 下划线
   - 当前光标位置：浅蓝背景 `#eef2ff`
   - 未输入字符：灰色 `#94a3b8`

### 三种模式

1. **按文本打字**：完整输入一段文本后结束，显示结果
2. **60秒计时**：固定 60 秒，时间到自动结束
3. **随机单词**：从词库随机抽取，持续打字直到手动停止

### 难度等级（按文本复杂度）

1. **简单**：常用字词、短句、无标点
2. **中等**：混合字词、有标点、中等长度
3. **困难**：长词、复杂标点、特殊字符

### 文本源

- 内置固定文本（中英文各 6-8 段）
- 随机单词库（中文 200+ 常用词，英文 300+ 常用词）

### 高级功能（Tab 展开）

1. **历史记录**：localStorage 保存最近 20 条记录
   - 包含：日期、模式、语言、速度、准确率、用时
2. **排行榜**：按速度排序 Top 10
3. **自定义文本**：用户输入自己的文本进行打字
4. **清除历史**：一键清除所有历史记录

### 控制栏说明

- **顶部左侧**：语言切换按钮（中文/English）
- **顶部右侧**：难度选择（简单/中等/困难）+ 自定义文本按钮

## 视觉反馈系统

### 打字时实时反馈

| 状态 | 样式 |
|------|------|
| 正确字符 | 绿色 `#10b981` |
| 错误字符 | 红色 `#ef4444` + 下划线 |
| 当前光标 | 浅蓝背景 `#eef2ff` |
| 未输入 | 灰色 `#94a3b8` |

### 完成后结果展示

- 保留颜色标记，不丢失打字过程
- 结果卡片：速度、准确率、用时、正确/错误字数
- 评语：根据速度给出等级
  - 中文：<30 字/分 "打字新手"，30-60 "打字达人"，>60 "打字高手"
  - 英文：<30 WPM "打字新手"，30-60 "打字达人"，>60 "打字高手"

### 动画

- 输入时字符平滑变色，无闪烁
- 统计数字变化时有轻微过渡
- 模式切换时无动画，直接切换

### 深色模式

- 支持 `prefers-color-scheme` 自动切换
- 暗色背景 `#1e293b`，浅色文字

## 技术实现

### 文件结构

单文件 `typing-test.html`，包含：
- HTML 结构
- CSS 样式（内联）
- JavaScript 逻辑（内联）

### 关键函数

1. `pickText(mode, lang, difficulty)` - 选择文本
2. `renderDisplay()` - 渲染文本显示
3. `updateStats()` - 更新统计
4. `checkComplete()` - 检查完成
5. `saveHistory()` - 保存历史
6. `loadHistory()` - 加载历史
7. `showResults()` - 显示结果

### 计算公式

- **中文 WPM**：正确字符数 / (时间 / 60)
- **英文 WPM**：正确字符数 / 5 / (时间 / 60)
- **准确率**：正确字符数 / 总输入字符数 × 100%
