# Game-for-me

一个可直接在浏览器中运行的小游戏合集。

## 游戏列表

| 游戏 | 文件 | 说明 |
|------|------|------|
| 舒尔特方格 | `schulte-grid.html` | 锻炼注意力和反应速度的经典方格训练 |
| 舒尔特方格完全版 | `舒尔特方格完全版.html` | 功能更丰富的舒尔特方格版本 |
| 反应测试 | `reaction-test.html` | 测试你的反应速度 |
| 五子棋 | `五子棋完全版.html` | 经典五子棋双人对弈 |

## 使用方式

直接用浏览器打开对应的 HTML 文件即可游玩，无需安装任何依赖。
